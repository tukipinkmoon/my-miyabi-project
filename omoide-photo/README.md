# おもいで写真 📸

**70代の方がAIアートを楽しめる、思い出の写真が動き出すアプリ**

思い出の写真が動き出して、話しかけてくれる。
白黒写真が動いた時の喜び。
亡くなったお父さん、お母さんが今の自分に話しかけてくれる。
遠く離れた孫が自分に話しかけてくれる。

そんな感動体験を、シンプルな操作で。

---

## 🚀 クイックスタート

### 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで http://localhost:3000 が自動的に開きます。

### ビルド

```bash
# 本番用ビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## 🎨 デザインコンセプト

### 高齢者向けUI
- **大きなボタン**: 最小60x60px
- **読みやすいフォント**: 最小18px
- **シンプルな言葉**: 専門用語を避ける
- **視覚的フィードバック**: 操作が分かりやすい

### カラーテーマ
- メインカラー: ピンク（温かみ）
- アクセント: グリーン（完了）、ブルー（情報）
- 背景: グラデーション（優しい雰囲気）
