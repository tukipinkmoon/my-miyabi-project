# おもいで写真 📸

**70代の方がAIアートを楽しめる、思い出の写真が動き出すアプリ**

---

## 🎯 コンセプト

白黒写真が動いた時の喜び。
亡くなったお父さん、お母さんが今の自分に話しかけてくれる。
遠く離れた孫が自分に話しかけてくれる。

そんな感動体験を、70代の方にも簡単に届けたい。

---

## ✨ 主な機能

### 1. 📱 超シンプルな操作
- アルバムから写真を選ぶだけ
- 難しい設定は一切なし
- 大きなボタン、わかりやすい言葉

### 2. 🤖 AIが自動で処理
- 写真の中の顔を自動で検出
- その人の雰囲気に合わせたメッセージを提案
- 音声も自動で生成

### 3. ✏️ メッセージは編集可能
- AIが提案したメッセージを見て
- 気に入ったらそのまま使う
- 自分の言葉に変えることもできる
- 「おかえり」「元気にしてる？」など、シンプルな言葉

### 4. 🎬 写真が動き出す
- 顔が動いて、話しかけてくれる
- リップシンク（口の動きが合っている）
- 白黒写真でもカラー化して動かせる（オプション）

### 5. 💾 保存・共有
- 動画として保存
- 家族や友達に簡単に共有
- 何度でも見返せる

---

## 🎨 ユーザー体験の流れ

```
1. アプリを開く
   ↓
2. 「写真を選ぶ」ボタンをタップ
   ↓
3. アルバムから思い出の写真を選ぶ
   ↓
4. AIが顔を検出（自動）
   ↓
5. AIがメッセージを提案
   「元気にしてる？いつも見守ってるよ」
   ↓
6. そのまま使うか、自分で編集
   ↓
7. 「動画を作る」ボタンをタップ
   ↓
8. 写真が動き出して、話しかけてくれる！
   ↓
9. 保存して、何度でも見返せる
```

## 💻 技術スタック

### フロントエンド
- **Phase 1（プロトタイプ）**: React（Web）
  - 素早くプロトタイプを作成
  - デスクトップでも確認可能
  - PWA化でスマホでも動作

- **Phase 2（本番）**: React Native
  - iOS/Android両対応
  - ネイティブアプリの体験
  - カメラ・アルバムとの連携が簡単

### バックエンド・AI
- **顔検出**: MediaPipe, face-api.js
- **顔アニメーション**:
  - D-ID API（有料、高品質）
  - LivePortrait（オープンソース）
  - Wav2Lip（オープンソース）
- **メッセージ生成**: Claude API
- **音声合成**:
  - ElevenLabs（自然な音声）
  - Google Cloud TTS（多言語対応）
  - OpenAI TTS

### ホスティング
- **フロントエンド**: Vercel, Netlify
- **バックエンド**: Cloudflare Workers, AWS Lambda
- **ストレージ**: AWS S3, Cloudflare R2

## 🔐 プライバシー・セキュリティ

### 重要な方針
1. **写真は保存しない**
   - 処理後すぐに削除
   - ユーザーのデバイスにのみ保存

2. **個人情報の保護**
   - 顔データは一時的な処理のみ
   - 第三者に共有しない

3. **透明性**
   - どのようにAIが使われているか説明
   - ユーザーが理解できる言葉で
